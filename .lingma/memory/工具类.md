# HelloAgents 工具系统完整指南

## 1. 核心概念

HelloAgents 提供了灵活的工具系统，支持多种 Agent 范式：
- **SimpleAgent**：简单对话 Agent，支持基于文本标记的工具调用。
- **ReActAgent**：推理与行动结合的 Agent，支持循环思考、行动和观察。
- **FunctionCallAgent**：利用 LLM 原生函数调用能力（如 OpenAI Function Calling）的 Agent。

## 2. 工具定义与注册

### 2.1 基础工具类 (Tool)
所有自定义工具应继承自 `hello_agents.tools.base.Tool`。

```python
from hello_agents.tools import Tool, ToolParameter
from typing import Dict, Any, List

class MyCustomTool(Tool):
    def __init__(self):
        super().__init__(
            name="my_tool",
            description="这是一个自定义工具的示例"
        )
    
    def run(self, parameters: Dict[str, Any]) -> str:
        # 实现工具逻辑
        return f"执行结果: {parameters}"
    
    def get_parameters(self) -> List[ToolParameter]:
        return [
            ToolParameter(
                name="input",
                type="string",
                description="输入参数",
                required=True
            )
        ]
```

### 2.2 使用装饰器自动展开 (@tool_action)
对于包含多个功能的复杂工具，可以使用 `@tool_action` 装饰器将其自动展开为多个独立子工具。

```python
from hello_agents.tools import Tool, tool_action

class MultiActionTool(Tool):
    def __init__(self):
        super().__init__(
            name="multi_tool",
            description="多动作工具",
            expandable=True  # 必须设置为 True
        )

    @tool_action("action_one", "执行第一个动作")
    def _do_first(self, query: str) -> str:
        """执行第一个动作"""
        return f"First action result for: {query}"

    @tool_action("action_two", "执行第二个动作")
    def _do_second(self, value: int) -> str:
        """执行第二个动作"""
        return f"Second action result for: {value}"
```

### 2.3 工具注册表 (ToolRegistry)
使用 `ToolRegistry` 统一管理工具。

```python
from hello_agents.tools import ToolRegistry

registry = ToolRegistry()

# 注册 Tool 对象
registry.register_tool(MyCustomTool())

# 注册可展开的工具（会自动展开为 multi_tool_action_one 等）
registry.register_tool(MultiActionTool(), auto_expand=True)

# 简便方式：直接注册函数
def simple_func(input_text: str) -> str:
    return f"Processed: {input_text}"

registry.register_function(name="simple_func", description="简单的处理函数", func=simple_func)
```

## 3. Agent 与工具集成

### 3.1 SimpleAgent (文本标记模式)
SimpleAgent 通过解析回复中的 `[TOOL_CALL:name:params]` 标记来调用工具。

```python
from hello_agents import SimpleAgent
from hello_agents.core.llm import HelloAgentsLLM

agent = SimpleAgent(
    name="SimpleBot",
    llm=HelloAgentsLLM(),
    tool_registry=registry,  # 传入注册表即可启用工具
    enable_tool_calling=True
)

response = agent.run("请帮我执行 simple_func")
```

### 3.2 ReActAgent (推理行动模式)
ReActAgent 遵循 `Thought -> Action -> Observation` 循环。

```python
from hello_agents.agents import ReActAgent

agent = ReActAgent(
    name="ReActBot",
    llm=HelloAgentsLLM(),
    tool_registry=registry,
    max_steps=5  # 限制最大思考步数
)

response = agent.run("分析当前的任务并给出结论")
```

### 3.3 FunctionCallAgent (原生函数调用)
FunctionCallAgent 利用 LLM 的原生工具调用接口，效率更高且更稳定。

```python
from hello_agents.agents import FunctionCallAgent

agent = FunctionCallAgent(
    name="FuncBot",
    llm=HelloAgentsLLM(),
    tool_registry=registry,
    max_tool_iterations=3
)

response = agent.run("查询最新的信息")
```

## 4. MCP 工具集成

HelloAgents 支持通过 `MCPTool` 接入 Model Context Protocol 服务。

```python
from hello_agents.tools import MCPTool

# 创建 MCP 工具实例
amap_mcp = MCPTool(
    name="amap",
    server_command=["uvx", "amap-mcp-server"],
    env={"AMAP_MAPS_API_KEY": "your_key"},
    auto_expand=True  # 自动将 MCP 提供的多个功能展开为独立工具
)

# 添加到任意支持工具的 Agent
agent.add_tool(amap_mcp)
```

## 5. 最佳实践

1. **选择合适的 Agent**：
   - 需要高稳定性和结构化输出时，优先选择 `FunctionCallAgent`。
   - 需要复杂逻辑推理和多步决策时，选择 `ReActAgent`。
   - 简单交互或轻量级工具调用时，选择 `SimpleAgent`。

2. **工具描述清晰化**：
   - `description` 字段直接影响 LLM 的选择准确率，务必详细描述工具的用途和参数要求。

3. **参数类型匹配**：
   - 在 `get_parameters` 中准确定义 `type`（string, integer, number, boolean），框架会自动进行类型转换。

4. **错误处理**：
   - 在 `run` 方法内部做好异常捕获，返回清晰的错误信息字符串，帮助 Agent 进行自我修正。
